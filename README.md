# hersey-site

Static personal site + Continuum analysis breakdown. Hosted on GitHub Pages.

- `index.html` — HERSEY landing page (Music × Machine)
- `get-down.html` — interactive Continuum track breakdown (Burnski & Kepler — Get Down)
- `get_down.mp3` — audio for the breakdown player
- `.nojekyll` — serve files as-is (no Jekyll processing)

Edit `index.html` / regenerate `get-down.html` (via the Continuum repo's
`deconstruction/build_get_down_public.py`), then commit + push to update the live site.
